# Ejercicios de JavaScript

## Descripción
Este repositorio contiene una colección de ejercicios desarrollados en JavaScript para practicar conceptos fundamentales del desarrollo web, incluyendo funciones tradicionales, funciones flecha (Arrow Functions), manipulación del DOM y manejo de eventos.

## Objetivos de aprendizaje
- Comprender la declaración y uso de funciones en JavaScript.
- Implementar funciones flecha (Arrow Functions).
- Manipular elementos HTML mediante el DOM.
- Gestionar eventos de teclado y mouse.
- Crear interacciones dinámicas en páginas web.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (ES6)

## Contenido
- Funciones tradicionales.
- Arrow Functions.
- Eventos de clic.
- Eventos de teclado.
- Modificación dinámica de estilos y colores.
- Manipulación del DOM.
